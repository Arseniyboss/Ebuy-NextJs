# Ebuy

## Description

Fullstack e-commerce website built with NextJS

## Technologies

- Language: **TypeScript**
- Framework: **NextJS**
- Styling: **Styled Components**
- Database: **MongoDB**
- API Tests: **Jest**
- E2E Tests: **Cypress**
